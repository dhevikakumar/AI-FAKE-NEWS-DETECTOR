The AI Fake News Detector is a web-based application that uses Machine Learning and Natural Language
Processing (NLP) to classify news as real or fake. Users can enter news text through a simple interface,
and the system processes it using text cleaning and TF-IDF vectorization before applying a Logistic Regression 
model to make predictions. It also provides a confidence score and highlights important keywords to explain 
the result, improving transparency and user trust. The project is developed using Python and Flask for the backend,
with HTML and CSS for the frontend, and uses libraries like scikit-learn, pandas, numpy, and nltk. It is easy to run
and can be further improved with advanced models, real-time data integration, and additional features.
